Adaptive Simpson Quadrature

A smart numerical integration technique that optimizes step size for accuracy.

\[ \int_{a}^{b} f(x) \, dx \approx S(a, b) = \frac{h}{3} \left[ f(a) + 4f\left(\frac{a+b}{2}\right) + f(b) \right] \]

Mathematical Discussion

Overview

Adaptive Simpson's Quadrature is an algorithm for numerical integration that improves upon the standard Simpson's Rule by using a recursive strategy. Unlike fixed-step methods, it focuses computational effort on areas where the function changes most rapidly.

How it Works

  1. Calculate the Simpson estimate for the whole interval \([a, b]\).
  2. Split the interval in half: \([a, m]\) and \([m, b]\), where \(m = (a+b)/2\).
  3. Calculate Simpson estimates for both halves.
  4. The Test: If the sum of the two halves is "close enough" to the whole estimate (within a tolerance threshold), the result is accepted.
  5. If not, the algorithm recursively repeats the process on each half with a stricter tolerance.

The Error Estimation

The criteria for stopping recursion is typically:

\[ |S(a, m) + S(m, b) - S(a, b)| \leq 15 \cdot \epsilon \]

Where \(\epsilon\) is the user-defined tolerance. The factor of 15 comes from the error term analysis of the fourth-order Simpson's Rule.

Step-by-Step Examples

Example 1

\[ \int_{0}^{1} e^{-x^2} dx \]

Goal: Approximate the Gaussian integral over [0, 1].

1. Initial Call: \(a=0, b=1, m=0.5\).
\(S(0, 1) = \frac{1}{6}[f(0) + 4f(0.5) + f(1)] \approx 0.74718\).

2. Subdivision: Split into [0, 0.5] and [0.5, 1].

3. Recursive Check: Estimates for halves are 0.38545 and 0.36137. Total: 0.74682.

4. Convergence: Difference is \(< \text{tolerance}\). Recursion stops.

Final Approximation: 0.74682

Example 2

\[ \int_{1}^{2} \ln(x) dx \]

Goal: Find the area under the natural log curve from 1 to 2.

1. Initial Call: \(a=1, b=2, m=1.5\).
\(S(1, 2) = \frac{1}{6}[\ln(1) + 4\ln(1.5) + \ln(2)] \approx 0.38583\).

2. Refinement: Because \(\ln(x)\) has a steep slope near 1, the algorithm subdivides the left interval [1, 1.5] more aggressively than the right.

3. Result: Sum of recursive calls yields high precision.

Final Approximation: 0.38629
